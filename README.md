# SlotBot
Telegram 中文老虎机 Bot

### How to build
```
git clone https://github.com/edward-p/SlotBot.git
cd SlotBot
./gradlew jar
```

### How to use
```
java -jar SlotBot.jar <TOKEN> <DATA_FILE_PATH>
```
e.g.

```
java -jar SlotBot.jar "00000000:AAHkGMEC8MbCdvr6IjJb4kEkP76bU8NOFQ8" $HOME/.cache/slotbot.cache
```
